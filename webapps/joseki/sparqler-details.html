SPARQLer

Query Processor: ARQ, the query processor for Jena. This currently (Nov 2004) implements all the working group features including the 4 result forms of SELECT, CONSTRUCT, DESCRIBE and ASK.

Server: Joseki, with modifications to match the working group decisions on use of HTTP and with a SPARQL handler. Current limitation: results are RDF/XML pending publication of the XML result format. Thsi also means that ASK queries aren't accessible.
