SPARQLer

This site provides a number of SPARQL services:

  • sparql : a general purpose SPARQL processor that processes graphs from the web.
  • books : a SPARQL processor that answers queries on small books database only.

Web interfaces:

  • SPARQL query validator
  • Form for queries on the books database
  • General purpose SPARQL processor

SPARQL is defined by 3 documents:

  • SPARQL query language
  • SPARQL protocol
  • SPARQL XML results format
