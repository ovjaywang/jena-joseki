Joseki is an HTTP and SOAP engine supports the SPARQL RDF Query language. SPARQL is the query language and protocol developed by the W3C RDF Data Access Working Group.

Joseki Features:

  • RDF Data from files and databases
  • HTTP (GET and POST) implementation of the SPARQL protocol
  • SOAP implementation of the SPARQL protocol
