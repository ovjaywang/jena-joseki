Joseki Downloads

Download Joseki

Joseki is open source, under a BSD-style license.

The Joseki server runs as a web application.

  • A complete system for running a standalone server. joseki-2.1.zip
    Includes source code, documentation and a web application server (Jetty)
  • The download includes an ant task to assemble Joseki as a web application, inckluding as a WAR file
  • (To come) Just the Joseki system with source code. and documentation but without supporting libraries. You will need a servlet container server and Jena.

The complete Joseki download area contains all Joseki releases.

You can also access the CVS repository for the project. Details are given in the SourceForge documentation.

Acknowledgements

Joseki relies on open source software:

  • Jena
  • Apache
    • Xerces
    • ORO
    • Tomcat
  • The Jetty web server
  • Doug Lea's concurrency package

Joseki requires Java 1.4 or later.
